# GOWDK Licensing

GOWDK is a mixed-license repository. The root `LICENSE` file contains the
Apache-2.0 license for the core project source code. Additional canonical
license texts live under `LICENSES/`.

## License Map

| Area | License or ownership | Notes |
| --- | --- | --- |
| GOWDK core, compiler, runtime, and addons | Apache-2.0 | See `LICENSE`. |
| Editor extensions | MIT unless otherwise stated | The VS Code extension declares MIT in its package metadata. |
| Examples, starters, and templates | MIT by default | Future files may use CC0-1.0 only when a nested license or metadata file says so. |
| Documentation, site, and content | CC BY 4.0 | Attribution: GOWDK contributors. |
| Generated apps and output | Owned by the app author | Not automatically licensed as GOWDK solely because GOWDK generated it. |
| Name, logo, and brand | Separate trademark policy | See `TRADEMARK.md`. |

## Canonical Texts

- Apache-2.0: `LICENSE`
- MIT: `LICENSES/MIT.txt`
- CC BY 4.0: `LICENSES/CC-BY-4.0.txt`
- CC0-1.0: `LICENSES/CC0-1.0.txt`

## Generated Output

Output generated by GOWDK from user-authored source files is owned by the
author of those source files and is not subject to the GOWDK project license
solely because it was generated with GOWDK.

Runtime files, helper libraries, or client assets copied from GOWDK remain
licensed under their original GOWDK license unless otherwise stated. Preserve
their license notices when distributing generated applications that include
copied GOWDK files or assets.

## Machine-Readable Metadata

The `.reuse/dep5` file declares the repository-level license mapping for tools
that understand Debian copyright/REUSE-style metadata.
