# v0.2 Release Checklist

This checklist tracks the v0.2 Public Truth and Release Trust bucket from
`docs/engineering/release-plan.md`. v0.2 is an experimental 0.x release target,
not a production-readiness claim.

## Status Rules

- `[x]`: current repository docs or observed release metadata support the item.
- `[ ]`: not done, not verified, or requires a GitHub-side/manual action.
- Release metadata and project board work may require GitHub UI or API actions
  outside the repository.
